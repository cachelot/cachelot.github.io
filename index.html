WHAT IS CACHELOT

Cachelot Library

If your application needs to cache some data and retrieve it from the cache with a speed of light, the only way is to store data right into application's memory. That's what cachelot library is.

One library with the simple interface. All you need is to provide fixed amount of memory for the cache and access or store 3 millions of items per second (depending on the CPU cache size). Maybe 3MOps doesn't sound like too big number, but it means ~333 nanosecond spent on a single operation while RAM reference costs ~100 nanoseconds.

Cachelot can work as a consistent cache returning an error when out of memory or evict old items to store new.

Distributed Cache Server

Think Memcached (it is Memcached compatible) but while original Memcached aims to serve hundreds of connections (bandwidth) Cachelot serves tens of them with higher RPS rate (latency).

Another good news: Cachelot is a single-threaded application, so you can run it consuming fewer resources (cheaper VPS hosts can be a huge saving).

And yet single Cachelot instance is faster than Memcached while serving a certain number of connections. Never the less, you're free to scale up horizontally by running more instances.

BENCHMARKS

Cachelot server performance using the memslap load generator.

The tests were performed on KVM virtual machines.

KVM Host:
Eight-core CPU Xeon E3-1240 / 16Gb RAM

Cachelot and Memcached VM:
2 vCPU / 2 Gb RAM

memslap VM:
4 vCPU / 8 Gb RAM

After running caching server (Cachelot or Memcached), it was warmed up by setting 1,600,000 items. Then memslap was executed 16 times with the same configuration: number of executions, concurrency, get or set test. Resulting time is average among all 16 runs.

PROJECT STATUS

Cachelot is still a 'work-in-progress', so there are some features that will be added in the nearest future.

  • Full set of Memcached commands: touch, append, prepend, stats
  • Binary protocol and UDP support
  • Library API and documentation
  • No library dependencies except STL
  • Microsoft Windows support
  • Binary and source packages
  • and more ...
